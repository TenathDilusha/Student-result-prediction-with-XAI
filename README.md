# Student Mental Health and Burnout Dataset

This is a large-scale synthetic dataset that models how academic pressure, lifestyle habits, and psychological well-being interact for students. It is designed for machine learning work in mental health analytics, predictive modeling, and behavioral insights.

## Key facts
- Size: 1,000,000 records.
- Features: 20 columns capturing stress, anxiety, sleep, academic performance, social factors, and related signals.
- Data type: Fully synthetic but crafted to preserve realistic relationships among variables.
- Intended users: ML practitioners, researchers, and data scientists exploring student mental health and burnout.
